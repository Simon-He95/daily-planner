说明:
  - 每天打开vscode，提醒你添加计划
  - 到达设定的计划开始时间，则提醒你开始计划
  - 可以在计划中添加描述，来记录这个任务中遇到的问题或者具体工作内容
  - 从周一到现在的计划生成一个周报(markdown)

## Feature
- 添加ai, 根据计划自动总结周报
- 计划支持增、删、改、查
